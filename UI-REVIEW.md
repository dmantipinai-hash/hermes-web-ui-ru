# UI Review — замечания Димы (2026-06-02)

## Критичные

### Адаптивность
- Верхняя панель (статусы, исполнители) тянется на весь экран
- Нельзя комфортно работать на пол-экрана
- Нужно: адаптивная вёрстка, которая работает при любом размере окна

### Drag & Drop
- Задачи нельзя перетаскивать мышью между колонками
- Нужно: grab-andle move задач между статусами

### Комментарии
- Неочевидно где писать комментарии (возможно функционал есть, но скрыт)
- Нужно: комментарии должны быть на виду, легко доступны

## Общее
- Текущий веб-интерфейс не отвечает требованиям — нужна доработка/переделка UI-слоя
- Нельзя углубляться в новые этапы пока UI не станет удобным
